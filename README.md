An interactive Power BI dashboard designed to analyze Amazon sales data, providing insights into sales performance, product trends, and regional distribution.


Objectives
  Analyze overall sales performance  
  Identify top-selling products and categories
  Track regional and state-wise sales
  Understand customer purchasing trends
  Monitor profit and revenue distribution

 Dashboard Features
    Sales Overview – Total revenue, profit, and quantity sold
    Product Analysis – Category and sub-category performance
    Regional Insights – Sales by region/state
    Time-Based Trends – Monthly/Yearly sales trends
    Interactive Filters – Dynamic filtering for better analysis

Tools & Technologies
  Power BI Desktop (.pbix)
  Power Query for data cleaning & transformation
  DAX (Data Analysis Expressions) for calculated measures
  Data Visualization techniques
